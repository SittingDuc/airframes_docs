# Compatibility

**Platforms**
- Embedded (ARM)
  - Raspberry Pi 3
  - Raspberry Pi 4
  - Rock Pi 4
- Personal Computers (x86)

**SDRs**
- RTL-SDR
- Airspy
- SDRplay RSP1A
- LimeSDR
- KerberosSDR

**Memory**
- Minimum: 2GB
- Recommended: 4GB

If you have had results outside of what is documented here, please submit a documentation update (click `Edit this page` below) or [contact us](mailto:kevin@airframes.io).
